# mrvl

#### Rodando o projeto!

  Só precisa instalar todas as dependências usando o yarn e rodar a aplicação com `yarn android`!
  Aplicação desenvolvida e testada utilizando um Asus max pro m2, funcionalidade no IOS não é garantida!
